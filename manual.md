### Requisitos
Para realizar alguna modificación o adición del contenido, debe iniciarse sesión en la página https://proteco.fi-b.unam.mx y tener permisos de becario.
Una vez dentro de la página dirigirse a la sección de **App Mobile** y seleccionar el apartado que se desea modificar.

### Crear nuevo curso<a name="id11"></a>

![image](https://user-images.githubusercontent.com/42527034/187041706-01097df6-043e-4bdf-a19a-7509361e0ff1.png)

Título: Nombre del curso.

Descripción: Descripción del curso, puede incluir promociones y emojis.

Fecha: Fecha y horario de impartición del curso.

Link: Link o URL de la imagen del curso, solo se permiten los formatos PNG y JPG, se puede obtener de la imagen publicada en Facebook de ese curso.
      Esta imagen es la que se mostrara en las aplicaciones móviles. Ejemplo de Link e imagen https://proteco.fi-b.unam.mx/img/publicaciones/Taller_Que_Computadora_debo_comprar.png

![image](https://user-images.githubusercontent.com/42527034/187043594-5df5d3a1-fe81-4e76-87db-8fc491bfdfa6.png)

Descripción de los campos.

Link: Es un link de un documento PDF almacenado en Google drive, es importante que el documento se guarde en Google drive con formato PDF, ya que si no es así no se podrá visualizar en la aplicación. Ejemplo de un link: https://drive.google.com/file/d/18i01JEeyCETtqapsRrsTGXtBQXuXeR0j/view?usp=sharing. 

Para obtener el link se debe abrir el documento PDF en Google drive, seleccionar la opción compartir y hacer clic en la opción **Copiar vinculo**.

Una vez modificado el link, seleccionar el botón **Actualizar Asesorías**.

Descripción de los campos.

Link: Es un link de un documento PDF almacenado en el servidor WEB de PROTECO, es importante que el documento se guarde en el Servidor de PROTECO con formato PDF, ya que si no es así no se podrá visualizar en la aplicación. Ejemplo de un link: https://proteco.fi-b.unam.mx/docs/convocatoria/Convocatoria_Gen43.pdf. 

Para obtener el link se puede consultar con el equipo encargado de la página Web.

Una vez modificado el link, seleccionar el botón **Actualizar Convocatoria**.
